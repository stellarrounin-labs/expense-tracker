# expense-tracker
A simple expense tracker application to manage your finances.

Sample solution for the [expense-tracker](https://roadmap.sh/projects/expense-tracker) challenge from [roadmap.sh](https://roadmap.sh/).

## How to run

Clone the repository and run the following command:

```bash
git clone https://github.com/stellarrounin-labs/expense-tracker
cd expense-tracker
```

Run the following command to build and run the project:

```bash
go build -o expense-tracker
```
